Willkommen zur Architektur-Karte

Diese Anwendung ist ein kleines Kanban-Board, das vier wichtige React-Konzepte kombiniert: Router, useReducer, Context und HTML5-Drag-and-Drop — alles persistiert im localStorage.

Klicke auf einen der farbigen Blöcke in der Karte, um eine ausführliche Erklärung mit Code-Beispielen zu sehen. Oder starte eine der Szenario-Touren links, um den Datenfluss als animierte Sequenz zu erleben.

Das Big Picture

App.tsx ist die Routing-Zentrale. Sie definiert drei Routen unter einem gemeinsamen Layout: /profile, /boards und /boards/:id. Der UserNameProvider umhüllt den Router, damit der gespeicherte Benutzername in jeder Route — besonders im TaskDialog — per useContext verfügbar ist.

Jede der beiden Daten-Seiten hält ihren eigenen State über useReducer: BoardOverview verwaltet die Board-Liste (ADD, DELETE), BoardDetail verwaltet die Tasks (UPDATE_BOARD_NAME, ADD_TASK, DELETE_TASK, UPDATE_TASK, UPDATE_TASK_STATUS). Jede Reducer-Änderung schreibt sofort in den localStorage, beim Mount wird der State über getBoards / getBoardById wiederhergestellt.

Das spannendste Detail ist die Drag-and-Drop-API: TaskCard setzt beim onDragStart die Spalten- und Task-ID in dataTransfer; jede BoardColumn liest sie im onDrop und dispatcht UPDATE_TASK_STATUS. Klicke auf den boardcolumn-Knoten in der Karte, um den Drop-Handler im Detail zu sehen.
