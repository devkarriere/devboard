Willkommen zur Architektur-Karte

Diese Anwendung ist eine kleine Nutzerverwaltung, die vier wichtige React-Konzepte kombiniert: States, Reducer, Context und Router.

Klicke auf einen der farbigen Bloecke in der Karte, um eine ausfuehrliche Erklaerung mit Code-Beispielen zu sehen. Oder starte eine der Szenario-Touren links, um den Datenfluss als animierte Sequenz zu erleben.

Das Big Picture

App.tsx ist die Zentrale. Sie haelt den globalen Zustand über useReducer, gibt ihn über UserContext an alle Kinder weiter und rendert gleichzeitig den Router — damit die einzelnen Views (Overview, Create, Edit) sowohl auf den State zugreifen als auch per URL aufgerufen werden koennen.

Ein besonderes Detail ist der Custom Hook useFormInput: Er wird in UserForm sieben Mal aufgerufen — und doch hat jedes Eingabefeld einen eigenen isolierten State. Das funktioniert, weil React jeden Hook-Aufruf intern als eigenen State-Slot speichert. Klicke auf den gruenen useFormInput-Knoten in der Karte, um genau zu verstehen, warum.
